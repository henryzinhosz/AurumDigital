# **App Name**: Aurum Digital

## Core Features:

- Dynamic Jewelry Catalog: An elegant grid display of jewelry products with specialized price handling for sales, fixed prices, or hidden quotes.
- Hidden Management Portal: A secure administrative dashboard accessible only via a secret URL for managing stock and categories.
- AI Collection Curater: A tool that uses generative AI to suggest refined product descriptions and metadata based on jewelry materials and style keywords.
- Integrated Category Logic: Full CRUD functionality for categories with the ability to associate multiple products seamlessly, persisted via database.
- Promotional Toggle Engine: Visual logic switches to mark products for the Main Cover, Hero Carousel, or Promotional Mode with dual-pricing display.
- Contact Integration: A streamlined communication bridge allowing customers to inquire about unpriced items directly via a subtle action button.

## Style Guidelines:

- The palette is anchored in 'Sunlit Sand' (#9C835E), a refined and versatile gold-toned primary color that evokes authenticity without being harsh.
- The background uses an ultra-clean 'Pearl Silk' (#FAF9F7), providing a warm off-white canvas that allows jewelry photos to pop.
- An accent color of 'Dusty Rosé' (#BA8B84) is used sparingly for promotional prices and call-to-action buttons to maintain delicacy.
- Headline and Title font: 'Playfair Display' (Serif) for a sophisticated, high-end fashion feel. Body font: 'PT Sans' (Sans-serif) for clean readability and warmth.
- Thin-line, minimalist icons with rounded caps to mirror the delicate nature of the jewelry products.
- An airy, spacious grid with generous margins and asymmetrical whitespace to suggest luxury and artisanal quality.
- Graceful fade-ins for product cards and soft horizontal pans for the Hero Carousel to create a serene shopping experience.
